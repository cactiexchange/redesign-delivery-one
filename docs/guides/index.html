Documentation Hub

The deployment ecosystem: a topology diagrammer feeds NetBox (the source of truth); an offline orchestrator deploys + validates a rack and records the result; reparsers-v2 turns that into the customer as-built. Pick your role below.

Role guides

System Administrator

Owns the platform & security posture
  • Ecosystem architecture & the three projects
  • NetBox source of truth + data model
  • License, vault & token security model
  • Secrets handling & system troubleshooting

Field Engineer

Runs the deployment on site (L2)
  • The check-in → deploy → check-out flow
  • Deployment kit & per-product cabling
  • Step control, generated passwords, validation
  • Troubleshooting on the floor

DevOps Administrator

Prepares deployments & maintains the toolchain
  • Seeding NetBox (workbook / JSON / interactive)
  • PEQ generation & minting engineer binaries
  • Hardware matrix & new-product onboarding
  • The as-built pipeline & CI checks
